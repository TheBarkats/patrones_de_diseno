Patrón Flyweight - Compartición de estado intrínseco

📚 ¿Qué es el Patrón Flyweight?

Flyweight optimiza memoria compartiendo datos comunes entre muchos objetos. Ideal para sistemas con miles de elementos similares.

Concepto Clave: Separar datos compartidos (tipo, color, textura) de datos únicos (posición).
