🔌 Adapter de Usuario - API Antigua a Moderna

Patrón Adapter - Integración de APIs incompatibles

📚 ¿Qué es el Patrón Adapter?

El patrón Adapter permite que interfaces incompatibles trabajen juntas. Es como un adaptador de corriente: convierte un formato de entrada a otro sin que el usuario sepa que existe la conversión.

❌ Problema:

API antigua devuelve:
{ nombre_completo, correo, fecha_creacion }

→
✅ Solución:

App espera:
{ fullName, email, createdAt }

ℹ️ Cómo Funciona el Adapter

  1. API Antigua devuelve datos con propiedades en español
  2. UserAdapter convierte cada propiedad:
    • nombre_completo → fullName
    • correo → email
    • fecha_creacion (string) → createdAt (Date)
  3. App Angular recibe datos listos para usar
  4. Ventaja: Si la API antigua cambia, solo cambias el Adapter
